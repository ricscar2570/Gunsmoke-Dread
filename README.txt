PER IL GAME MASTER:
-------------------

PREPARAZIONE MINIMA (1 ora):
- Leggi "Come Leggere Questo Quickstart" nel file principale
- Stampa la Quick Reference (ESSENZIALE!)
- Leggi Atto I dell'avventura (Part 2)
- Scorri velocemente il Bestiario (Part 3)

- Tieni sempre visibile la Quick Reference (digitale o stampata)

PER I GIOCATORI:
----------------

1. Apri gunsmoke-dread-quickstart.html
2. Vai alla sezione "Personaggi"  
3. Scegli tra: Jack Malone (Pistolero), Sarah Blackwood (Predicatrice),

MATERIALI NECESSARI:
--------------------

- Questi file HTML (aperti in browser)
- Dadi poliedrici: 2d12, 1d4, 1d6, 1d8, 1d10, 1d20
- Token/segnalini per tracciare Speranza e Paura
- Stampante (opzionale ma raccomandato per Reference e Sheets)


COSA SERVE SAPERE SU DAGGERHEART:
----------------------------------

Il quickstart assume conoscenza base del sistema Daggerheart:
- Tiri con 2d12 (1 Speranza + 1 Paura)
- Sistema risorse Speranza/Paura
- Combattimento base (Evasione, HP, Armatura)
- Connessioni tra personaggi

Se non conosci Daggerheart: il sistema si impara giocando! Chiedi al GM 
una spiegazione rapida (5 minuti) delle meccaniche base all'inizio della 
sessione.
